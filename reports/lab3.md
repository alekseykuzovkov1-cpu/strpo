# Лабораторная №3

## Ход работы

### 1. Для начала
#### 1.1. Мои программы
* редактирование текстовых файлов - Microsoft Word
* программированиe на главном языке программирования - VS Code
* выполнение команд в консоли - Git Bash
* навигация по файловой системе и манипуляция с файлами - Windows Explorer
* работа с гитом - Git Bash, VS Code
#### 1.2. Альтернативы моим программам
* редактирование текстовых файлов - LibreOffice (для простого редактирования текста LibreOffice работает быстрее и дает больше контроля над базовыми операциями вроде поиска и выделения)
* программированиe на главном языке программирования - Sublime Text (быстро запускается, более минималистичный интерфейс, однако для меня более предпочтительным оказался VS Code, т. к. он более простой для работы)
* выполнение команд в консоли - Git Bash
* навигация по файловой системе и манипуляция с файлами - Windows Explorer
* работа с гитом - Git Bash, VS Code
